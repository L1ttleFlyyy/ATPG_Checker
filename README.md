# ATPG_Checker
Automated script and golden results

# Build

## clone and copy your simulator
```
git clone git@github.com:L1ttleFlyyy/ATPG_Checker.git
cd ATPG_Checker
cp <path to your simulator> ./simulator
```
# Usage

```
# follow the instruction and enjoy!
./ultChecker
```
#Example
```
>> ./ultChecker
```
